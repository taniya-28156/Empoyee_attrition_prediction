# Employee Attrition Prediction

This project predicts whether an employee is likely to leave the company or not using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Features
- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Machine Learning Model Training
- Employee Attrition Prediction

## Dataset
IBM HR Analytics Employee Attrition Dataset
